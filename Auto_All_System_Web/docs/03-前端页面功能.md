## 🎨 UI/UX 设计特点

### 设计原则

1. **简洁直观**
   - 清晰的信息层级
   - 明确的操作流程
   - 简洁的界面设计

2. **现代美观**
   - 渐变色设计
   - 卡片式布局
   - 阴影和动画效果

3. **响应式布局**
   - 适配桌面端 (>1920px)
   - 适配笔记本 (1366px - 1920px)
   - 适配平板 (768px - 1366px)
   - 适配手机 (<768px)

4. **交互友好**
   - 实时反馈
   - Loading状态
   - 操作确认
   - 错误提示

### 颜色方案

- **主色调**: `#409EFF` (蓝色)
- **成功色**: `#67C23A` (绿色)
- **警告色**: `#E6A23C` (橙色)
- **危险色**: `#F56C6C` (红色)
- **信息色**: `#909399` (灰色)
- **背景色**: `#F0F2F5` (灰白色)

### 组件库

使用 **Element Plus** 组件库：
- el-button (按钮)
- el-table (表格)
- el-form (表单)
- el-dialog (对话框)
- el-card (卡片)
- el-tag (标签)
- el-badge (徽章)
- el-menu (菜单)
- el-pagination (分页)
- el-message (消息提示)
- 等等

## 📊 数据展示规范

### 金额显示

- 格式: `¥1,234.56`
- 精度: 保留2位小数
- 千分位分隔符

### 时间显示

- 绝对时间: `2026-01-16 15:30:45`
- 相对时间: `3分钟前`, `2小时前`, `昨天`

### 状态标签

| 状态 | 颜色 | 说明 |
|------|------|------|
| pending | gray | 待处理 |
| running | blue | 执行中 |
| success | green | 成功 |
| failed | red | 失败 |
| cancelled | gray | 已取消 |

### 百分比显示

- 进度条: 0-100%
- 成功率: 保留2位小数
- 颜色渐变: 低（红）→ 中（黄）→ 高（绿）

## 🔄 页面交互

### Loading状态

```vue
<el-button :loading="loading">
  提交
</el-button>
```

### 操作确认

```javascript
ElMessageBox.confirm('确定要删除吗？', '警告', {
  confirmButtonText: '确定',
  cancelButtonText: '取消',
  type: 'warning'
})
```

### 成功提示

```javascript
ElMessage.success('操作成功！')
```
